## Getting Started
Steps to get a local copy up and running follow these simple steps.

### Installation
1. Clone the repo:
  `git clone https://github.com/josmadev/alea-frontend-test.git`
2. Install NPM packages
  `npm install`
3. Run the app with:
  `npm run start`
